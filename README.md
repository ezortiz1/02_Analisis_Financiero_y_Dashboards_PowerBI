# Dashboard de Inteligencia Financiera: Análisis de Estados Financieros

Este proyecto consiste en el desarrollo de un ecosistema de visualización de datos financieros utilizando **Power BI** y **Excel**. El objetivo principal es transformar datos contables estáticos (Balances Generales y P&G) en una herramienta dinámica para el análisis de rentabilidad y estructura de capital.

## Alcance del Proyecto
* **Periodo de análisis:** 2021 - 2022.
* **Fuentes de datos:** Reportes anuales de Balance General (BG) y Pérdidas y Ganancias (P&G).
* **Herramientas:** Power BI Desktop, Power Query, DAX y Excel.

## Habilidades Técnicas Aplicadas
* **ETL (Power Query):** Consolidación de múltiples archivos de Excel, limpieza de encabezados y tipado de datos.
* **Modelado de Datos:** Creación de un modelo de estrella vinculando tablas de hechos (Saldos) con tablas de dimensiones (Catálogo de Cuentas).
* **Cálculos con DAX:** Creación de medidas personalizadas para:
    * Margen de Utilidad Bruta y Operacional.
    * Variaciones interanuales (YoY).
    * Cálculo automático de impuestos y utilidad neta.
* **Data Visualization:** Diseño de paneles interactivos con filtros por año, mes y tipo de cuenta.

## Estructura de archivos
* `Dashboard/`: Contiene el archivo `.pbix` con el reporte interactivo.
* `Data_Source/`: Archivos fuente en Excel utilizados para el modelo.
